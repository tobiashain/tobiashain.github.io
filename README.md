# Portfolio – Tobias Hain

Personal developer portfolio showcasing my work, skills, and a preview of [TobiOS](https://tobiashain.github.io/TobiOS/), a fully interactive browser‑based operating system.

## 🚀 Tech Stack

- **React 19** + **TypeScript**
- **Motion** (Framer Motion) – animations & scroll effects
- **SCSS** – component‑scoped styling with CSS variables
- **Vite** – fast builds & hot reload

## ✨ Features

- Animated hero section with dynamic word splitting & particle background
- Career timeline with work experience, education, and side projects
- Interactive skill chips grouped by proficiency (advanced / familiar)
- Project gallery with expandable “View all” and modal detail windows
- Image lightbox within project modals (click any thumbnail)
- “TobiOS” teaser – a Windows‑style desktop preview with taskbar, icons, and 3D tilt
- Fully responsive layout (mobile‑optimized header & grid)
